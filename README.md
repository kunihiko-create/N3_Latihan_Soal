# JLPT N3 練習アプリ

過去問の練習と、過去問をもとにした類似問題の練習ができる、ビルド不要の静的Webアプリです。
（聴解は現在ふくまれていません。）

## 起動方法

**`index.html` をブラウザで直接開くだけで動きます**（ダブルクリックでOK）。
データは `data/bundle.js` にまとめてあり、`<script>` で読み込むので、
ローカルサーバーは不要です。

GitHub Pages にそのまま置いても動きます（リポジトリにアップロードして Pages を有効化するだけ）。

簡易サーバーで開きたい場合は次のようにします（任意）。

```bash
cd "/Users/takadakunihiko/LIBE-SKILLUP4/JLPT_SOAL/JLPT-N3-App"
python3 -m http.server 8000
# → http://localhost:8000
```

## できること

- **過去問練習**：回 → セクション（文字・語彙／文法／読解）を選び、本番と同じ問題を順に解く。
- **類似問題練習**：セクションを選び、過去問をもとに作った類似問題をランダムに解く。
- 1問ごとに正誤判定・正解表示・解説、最後に得点と「間違えた問題の見直し」。
- **多言語UI**：ヘッダーの言語セレクタで「日本語のみ／インドネシア語のみ／両方併記」を切替（選択は保存）。
  メニュー・ボタン・指示文・解説をインドネシア語化。**問題文・選択肢は学習対象のため日本語のまま**。
- 漢字読み・言い換えなどの問題では、対象語に**下線**を表示。

## フォルダ構成

```
JLPT-N3-App/
  index.html         アプリ本体（1ページ）
  css/style.css
  js/app.js          画面遷移・出題・採点
  js/i18n.js         UI文言・指示文の翻訳（日本語/インドネシア語）
  js/data.js         データ読み込み（bundle.js から取得）
  data/
    index.json       収録している試験・類似セットの一覧
    past/            過去問（例: 2023-07.json）
    similar/         類似問題（例: 2023-07.json）
  tools/render_pdf.py 開発用: PDFページ→PNG変換（抽出作業の補助。実行時は不要）
```

## データの増やし方

1. `data/past/<YYYY-MM>.json` を既存ファイルと同じ形式で追加。
2. `data/index.json` の `past`（必要なら `similar`）に1行追加。
3. **`python3 tools/build_bundle.py` を実行して `data/bundle.js` を再生成**。

（アプリは `data/bundle.js` だけを読み込みます。JSON を編集したら必ず手順3を実行してください。）

データ形式は `data/past/2023-07.json` を参照してください。
`answer` は選択肢の番号ではなく **0始まりのインデックス**（1番＝0, 2番＝1 …）です。

インドネシア語表示について：
- 指示文（`instruction`）の訳は `js/i18n.js` の `INSTRUCTION_ID`（日本語文の完全一致 → 訳）で管理。新しい指示文を追加したらここに1行足す（未登録なら日本語のみ表示）。
- 解説の訳は各問題の `explanation_id` フィールドに入れる（無ければ日本語のみ表示）。
